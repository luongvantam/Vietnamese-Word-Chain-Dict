# Vietnamese Word Chain Dictionary

Tuyển tập các từ tiếng Việt dạng `.txt` chất lượng cao. Dữ liệu đã được lọc bỏ các từ đơn, tập trung vào các từ ghép có nghĩa để phù hợp với quy tắc của trò chơi nối từ.

## Thông tin
* [Từ điển Winstonleedev](https://github.com/winstonleedev/tudien)
* [Từ điển Underthesea](https://github.com/undertheseanlp/dictionary/)

**Thông số kỹ thuật:**
- Tổng số lượng từ: `~57000` (Đang trong thời gian cập nhập)
- Loại từ: Từ ghép (2 chữ)
